## ⚠️ 安全须知

1. **安装前扫描**：使用 `clawhub install skill-vetter` 安装安全扫描工具
2. **检查源码**：安装前查看 Skill 的 SKILL.md 和脚本内容
3. **验证发布者**：优先选择下载量高、有 GitHub 仓库的 Skill
4. **权限最小化**：只给 Skill 必要的环境变量和权限
5. **定期更新**：使用 `clawhub update --all` 保持 Skill 最新
6. **查看 VirusTotal 报告**：ClawHub 已集成 VirusTotal 扫描，安装前查看安全报告
